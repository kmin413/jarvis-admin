# 자비스 AI 비서 프로그램

일정 관리, 할 일 목록, 메모 작성 등 다양한 기능을 제공하는 AI 비서 프로그램입니다.

## 주요 기능

### 🤖 명령 센터
- 자연어로 명령을 입력하여 작업 수행
- 할 일, 일정, 메모를 음성/텍스트로 추가 가능
- 예시: "할일 회의 준비하기 추가", "일정 내일 오후 3시 미팅"

### 📅 일정 관리
- 월별 캘린더 뷰
- 일정 추가, 수정, 삭제
- 날짜별 일정 목록 확인

### ✅ 할 일 목록
- 할 일 추가 및 완료 체크
- 우선순위 설정 (낮음/보통/높음)
- 필터링 기능 (전체/진행 중/완료)

### 📝 메모
- 메모 작성 및 편집
- 태그 기능
- 검색 기능

### ⚙️ 설정
- 데이터 내보내기/가져오기
- 자동 저장 설정
- 데이터 관리

## 기술 스택

- React 18
- TypeScript
- Vite
- date-fns
- Lucide React (아이콘)

## 설치 및 실행

### 필수 요구사항
- Node.js 18 이상
- npm 또는 yarn

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

브라우저에서 `http://localhost:3000`으로 접속하세요.

### 빌드
```bash
npm run build
```

### 미리보기
```bash
npm run preview
```

## 데이터 저장

모든 데이터는 브라우저의 로컬 스토리지에 저장됩니다. 데이터를 백업하려면 설정 메뉴에서 "데이터 내보내기" 기능을 사용하세요.

## 사용 방법

1. **명령 센터**: 자연어로 명령을 입력하여 빠르게 작업을 수행합니다.
2. **일정 관리**: 캘린더에서 날짜를 클릭하여 일정을 추가합니다.
3. **할 일 목록**: 할 일을 추가하고 완료 체크를 합니다.
4. **메모**: 중요한 정보를 메모로 저장하고 태그로 관리합니다.
